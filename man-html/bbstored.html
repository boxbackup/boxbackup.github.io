Name

bbstored — Box Backup store daemon

Synopsis

bbstored [config-file]

Description

bbstored runs on a central server. Clients running bbackupd connect to the server and upload files.

The only argument is optional and specifies a non-default configuration file. By default it will look for the configuration file as /etc/box/bbstored.conf.

Files

/etc/box/bbstored.conf

See Also

bbstored.conf (5) , bbstored-config (8) , raidfile-config (8) , raidfile.conf (5)
